# Flex & Grid Workshop
CSSのFlexboxとGridレイアウトを学ぶための実践です。

## 内容
- 'flex/': Flexboxの課題と解答
- 'grid/': Gridレイアウトの課題と解答

## 課題の進め方
1.'starter/index.html'をエディタで開いてください
2.コメントを参考にしながらCSSを追加・修正してレイアウトを完成させてください
3.完成したら、'solution/index.html'と見比べて確認しましょう

---

### Flexのゴール
- ナビゲーションを横並びにする
- 中央に配置する

### Gridのゴール
- 3列のカードレイアウトを作る
- 均等なスペースを保つ
